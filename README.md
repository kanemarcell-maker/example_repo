**"finance calculators"**

Allows the user to access two different financial calculators that calcuate the additional interest that is earned or owed where an investemnt or bond is issued:

**Investment calculator:** calculates the interest earned based on specific uer input
    - interest rate (%)
    - initial deposit ($)
    - time horizon (years)

**Bond repayement formula:** calculates the interest owed on debt for a house. User input consists of:
    - Present value of the house ($)
    - monthly interest rate (%)
